GPPLib Documentation

Glider Path Planning Library

Written by Arvind Pereira,
University of Southern California, Los Angeles, USA

Glider Path Planning Library contains path planning tools for Autonomous Underwater Vehicles. Many of the tools in it might be useful and applicable to other robots etc. This file links to the documentation for this library. To have it point to the actual documentation you will need to run make in this folder.

  • Python documentation for Glider Path Planning Library may be found here.
  • GPP in CPP documentation for the Boost-Python loadable modules may be found here.

    GPP in CPP dependencies

    This library has some dependencies. For the 'GPP in CPP' part, you will need to have
  • A C++ compiler such as g++ or gcc
  • Python
  • Boost
  • SetupTools (to build and install gpp_in_cpp)
  • Doxygen (for Documentation)

    GPPLib (Python) dependencies

    The Python part of GPPLib requires:
  • Python v2.6 (or higher)
  • Numpy v1.6 (or higher)
  • Scipy v1.0 (or higher)
  • Matplotlib v.0.9 (or higher)
  • NetworkX v1.6 (or higher)
  • SetupTools (to build and install gpplib)
  • Sphinx (for Documentation)
